## Definition of hybrid

a combination of multiple species

## Examples

- fox + wolf = [folf](./folf)
- mammal + robot = [protogen](./protogen)
- fox + shark = [manokit](./manokit)
- lion + tiger = liger
- wolf + dog = wolfdog
- wolf + husky = wolfsky or wusky
- wolf + tiger = wolger
- cat + rabbit = cabbit
- (jack)rabbit + antelope = jackalope
- fox + raccoon = foxcoon
- lion + goat + snake = chimera
- eagle + lion = gryphon
